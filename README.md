# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

*リモートリポジトリは、複数人で共有して編集できるような、ネット上にあるリポジトリ。

*ローカルリポジトリは、開発者が一人で操作するために、自分のPC上に配置しているリポジトリ。

## プッシュとマージの違いは何でしょうか？

*保存した内容をリモートリポジトリにアップロードするのが、プッシュ。

*別のブランチの作業内容、変更履歴をブランチに取り込むのが、マージ。

## コミットとプッシュの違い

*コミットは、ローカルリポジトリに変更内容を記録する操作。

*プッシュは、ローカルリポジトリに記録された変更履歴をリモートリポジトリにアップロードする操作。

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

*変更内容が細かく分かるように。別の変更と混ざるような紛らわしい記述は避ける。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

*コードレビューの前にマージするかどうか。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

*先にマージされた変更内容を取り込む

*後にマージしようとしている変更内容を取り込む

*どちらの変更内容も取り込む
